# CPA 智能题库 Agent

一个基于 RAG（检索增强生成）的 CPA 备考智能学习助手。

## 技术栈
- Python + LangChain
- DeepSeek / 智谱 API（LLM）
- Chroma（向量数据库）
- Gradio（前端界面）
- SQLite（错题本 V2）

## 项目阶段
- Phase 1（Week 1-4）：编程地基 + LLM API 入门
- Phase 2（Week 5-9）：Agent / RAG 核心
- Phase 3（Week 10-13）：Multi-Agent 升级 + 求职启动

## 启动
```bash
pip install -r requirements.txt
python src/main.py
```
